# FlowDesk 🎫

Sistema de chamados corporativo desenvolvido com ASP.NET MVC.

## Status
🚧 Em desenvolvimento — v0.1.0

## Tecnologias
- ASP.NET MVC (C#)
- MySQL
- Bootstrap + CSS customizado

## Funcionalidades implementadas
- Cadastro e login de usuários
- Autenticação com cookies
- Diferenciação de perfis (Admin/Usuário)
- Dashboard com sidebar responsiva

## Em breve
- CRUD completo de chamados
- Dashboard com métricas
- Sidebar com abrir/fechar
